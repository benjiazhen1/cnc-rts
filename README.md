# C&C RTS - 致敬经典即时战略游戏

基于EA开源的红警/泰伯利亚黎明源码学习的RTS游戏

## 技术栈
- **引擎**: Godot 4.x
- **语言**: GDScript
- **目标平台**: Web + Steam (Windows)

## 游戏类型
即时战略（RTS）- 造基地、爆兵、推塔
